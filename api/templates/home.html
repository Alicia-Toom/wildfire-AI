Welcome to Wildfire AI

Wildfire ai is a api interface for detecting forest wildfires. The problem with wildfires is that they can spring up when you aren't looking.
This a concern for us, and our solution is to create this platform for feeding images and videos to our machine learning models.
These machine learning models will detect and monitor for any wildfire. If any are detected a intelligent system of alerts
can be setup on this platform to alert the wanted authority and concerned individuals.

Our goal

Our goal is a platform that solve the problem for human monitoring of wildfires. We will do this by take developing a intelligent and
optimized system architecture both with machine learning fire detection and also with alerting.
